Detailed Description

This file implements the functions for creating nodes of the Abstract Syntax Tree (AST).

It provides the concrete implementation for each make_*_node function declared in AST.h. These functions are responsible for allocating memory for a new AST node and initializing its fields with the appropriate values and type information. These nodes are then used by the parser to build the full AST.
